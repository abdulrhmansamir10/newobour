Primary action control — use for CTAs, form submits, and navigation actions across NewObour.

```jsx
<Button variant="primary" size="md">ابحث في الكشوف</Button>
<Button variant="accent" iconStart={<i data-lucide="badge-check" />}>عرض موثّق</Button>
<Button variant="outline">تواصل مع البائع</Button>
```

Variants: `primary` (Deep Navy, default), `accent` (Royal Gold — verified/high-emphasis), `outline`, `ghost`, `danger`. Sizes `sm | md | lg`. Props: `block`, `loading`, `disabled`, `iconStart`, `iconEnd`. RTL-first — icons flow with text direction.
